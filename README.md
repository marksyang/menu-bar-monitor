# Menu Bar Monitor

A lightweight macOS menu bar application that monitors system metrics in real-time using SwiftUI.

## Features

- 📊 **Real-time Monitoring**: Track CPU, GPU, and Memory usage.
- 🌡️ **Thermal Insights**: View CPU temperature and thermal pressure states.
- 🌀 **Fan Speed**: Monitor active fan RPMs.
- 🎨 **Customizable Modes**:
  - **Compact**: Minimal stats display.
  - **Standard**: Focused temperature tracking.
  - **AI Mode**: Comprehensive view including Swap usage and Pressure metrics.
  - **Icon Mode**: Minimalist icon-only display.
- ⚠️ **Alert States**: Automatically highlights critical thermal or memory pressure in the menu bar.

## Requirements

- macOS 13.0+ (Ventura)
- Xcode 15.0+
- Swift 5.9+
